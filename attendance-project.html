AI-Driven Real-Time Attendance Monitoring System

By: Prabhala Sai Sirisha Devi

Department / College: CSE - Honors, KL Deemed to be University

Abstract

  • A smart attendance system using AI for secure and accurate tracking.
  • Combines multi-frame face recognition and gait-based motion analysis.
  • Stores data securely with offline cache-like memory.
  • Prevents misuse and increases reliability over existing methods.

Objectives

  • ESP32-CAM for facial data capture and recognition.
  • Sync attendance with Firebase in real time.
  • Fetch data into a Django-based web interface for management.
  • Add gait recognition for uniqueness and accuracy.
  • Offline attendance storage until network sync.
  • Multi-factor validation using face + gait + device location.

Problems in Current Systems

  • Smartphone misuse: time/date manipulation, tech illiteracy.
  • Biometric spoofing via fake fingerprints or proxy attendance.
  • Fixed, costly external biometric devices.

  • Static face databases without remote updates.
  • No motion validation or liveness detection.

Planned Solutions

  • Local cache for offline attendance storage.
  • Admin upload of facial data to Firebase for remote sync.

Conclusion
